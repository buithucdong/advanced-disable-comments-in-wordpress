Disable comments on posts, pages, WooCommerce products, and via API with flexible options.

== Description ==

Advanced Disable Comments cung cấp giải pháp linh hoạt để tắt bình luận trên website WordPress của bạn. Plugin cho phép tắt bình luận theo từng loại nội dung hoặc trên toàn bộ website.

**Tính năng chính:**

* Tắt bình luận ở mọi nơi
* Tùy chọn tắt bình luận cho:
  * Bài viết (Posts)
  * Trang (Pages)
  * Sản phẩm WooCommerce (Products)
* Bảo vệ khỏi bình luận qua API:
  * XML-RPC
  * REST API

== Installation ==

1. Tải plugin lên thư mục `/wp-content/plugins/`
2. Kích hoạt plugin qua menu 'Plugins' trong WordPress
3. Vào 'Cài đặt' -> 'Disable Comments' để cấu hình

== Frequently Asked Questions ==

= Plugin có ảnh hưởng đến SEO không? =

Không, việc tắt bình luận không ảnh hưởng tiêu cực đến SEO. Thực tế, nó có thể giúp giảm spam và cải thiện trải nghiệm người dùng.

= Tôi có thể tắt bình luận trên một số bài viết cụ thể không? =

Plugin hiện tại chỉ hỗ trợ tắt bình luận theo loại nội dung. Để tắt bình luận trên bài viết cụ thể, bạn có thể sử dụng tính năng có sẵn của WordPress.
